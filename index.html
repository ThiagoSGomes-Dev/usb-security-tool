Installation

01
Clone the repo onto your USB drive
Make sure you are cloning directly into the USB drive root.
git clone https://github.com/youruser/usb-security-tool E:\
02
Download Python Embeddable
Go to python.org/downloads/windows and download Windows embeddable package (64-bit). Extract the contents into the python\ folder on the drive.
E:\ └── python\ ├── python.exe ← must be here └── ...
03
Run the installer
Open CMD, navigate to the drive and run INSTALL.bat. It handles everything automatically.
E: INSTALL.bat [1/4] Checking drive type... OK - USB removable drive detected: E: [2/4] Checking portable Python... OK - python\python.exe found. [3/4] Installing dependencies... done. [4/4] Hiding system files... OK - all files hidden. Done! Opening secure terminal...

Usage

$ usb
Opens the interactive menu. Choose encrypt, decrypt, or inspect from a numbered list.
$ usb --enc <path>
Encrypts all files in the target directory. Already encrypted files are skipped automatically. Original files are deleted after encryption.
$ usb --dec <path>
Decrypts all .enc files in the target directory using your password. Wrong password = instant failure, no data exposed.
$ usb --inspect <path>
Shows how many files are encrypted vs plaintext without modifying anything.
$ usb --help
Displays the ASCII banner with all available commands and usage examples.

Security Model

Component Detail
Encryption AES-256-GCM — authenticated encryption, detects tampering
Key derivation PBKDF2-SHA256 with 480,000 iterations — brute force resistant
Salt 16 bytes, random per file — prevents rainbow table attacks
Nonce 12 bytes, random per file — identical files produce different ciphertext
Key storage Never stored — derived at runtime, lives only in RAM
Network Zero — fully offline after first-time setup
Backdoor None — wrong password = unrecoverable data
